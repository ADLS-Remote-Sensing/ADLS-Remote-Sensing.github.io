



Introduction to Methods


Methods are functions tied to specific objects. 

```js
var myname = "Nils Ratnaweera"

// in the console:
myname.length
```

```js
var myage = 34
// in the console
myage.toFixed(2) // 2 decimal places
```


```js
var myname = "Nils Ratnaweera"
// in the console
myname.split(" ")

myname.split("").reverse()

myname.split("").reverse().join("")
```
